# Free tools docs

> Two open-source (MIT) checks for how search and AI crawlers see a publication —
> free, live now, no sign-in.

## Web Bloat Checker

**What it is** — audits a page for weight and for third-party trackers /
reader-data leakage, and grades it.

**How to use it**
1. Open [tools.onlinejourno.com](https://tools.onlinejourno.com/) → Web Bloat Checker.
2. Enter a public page URL.
3. Read the weight breakdown and the privacy/tracker findings.

**Why it matters** — heavy, tracker-laden pages cost readers and search
standing. This shows the cost in plain numbers.

## Crawl-Budget Analyser

**What it is** — estimates how efficiently search and AI crawlers can reach a
site's content given its structure and response behaviour.

**How to use it**
1. Open [tools.onlinejourno.com](https://tools.onlinejourno.com/) → Crawl-Budget Analyser.
2. Enter the site URL.
3. Read where crawl budget is being wasted and what to fix.
