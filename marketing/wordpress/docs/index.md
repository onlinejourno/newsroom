# Documentation

OnlineJourno is an open, fair-source suite of tools for the digital newsroom —
from a journalist, for journalists. This is the reference: what each tool does,
how to use it, and how to run it yourself.

## Start here

New to OnlineJourno? The suite splits into three kinds of tool:

- **Daily desk tools** you sign into — [Newsroom](/docs/newsroom/) (the full
  workspace) and [Daybook](/docs/daybook/) (the predictive calendar).
- **Point checks** you run on a URL or a feed — [Galley](/docs/galley/) (grade
  one story), [Frontmatter](/docs/frontmatter/) (audit a whole feed), and the
  [free tools](/docs/tools/) (page-bloat + crawl-budget).
- **A public read** — [Pulse](/docs/pulse/), a curated digest of undervalued
  Indian journalism.

## The tools

| Tool | What it does | Licence |
|------|--------------|---------|
| [Newsroom](/docs/newsroom/) | The agentic workspace: watch the record → contextualise → ranked beat briefs → pre/post-publish diagnostics | Fair-source (FSL) |
| [Daybook](/docs/daybook/) | Predictive editorial calendar — see the story coming while there's time to report it | Fair-source (FSL) |
| [Galley](/docs/galley/) | Graded report for a single story — where it's strong, where it loses readers, the fix | Fair-source (FSL) |
| [Frontmatter](/docs/frontmatter/) | Audit a whole feed for the merit–reach gap — deserving work that isn't travelling | Fair-source (FSL) |
| [Free tools](/docs/tools/) | Page-bloat + tracker auditor, crawl-budget analyser | Open source (MIT) |
| [Pulse](/docs/pulse/) | Curated digest of high-merit, low-reach Indian stories | — |

## Run it yourself

Every tool is source-available and built to self-host. See
[Self-hosting](/docs/self-hosting/) for the install path, configuration, and the
module contract for extending it.
