# Contributing

Thanks for your interest. OnlineJourno Newsroom / Platform is **proprietary
software**. The source is made available for reference, audit, and review only.
External contributions are generally not accepted.

## What you can do

- **Report bugs** or security vulnerabilities via the process in
  [`SECURITY.md`](SECURITY.md) — do not open a public issue for security matters.
- **Request a commercial licence** by emailing licensing@onlinejourno.com.

## What you may not do without a written licence

- Use the software in production.
- Modify, distribute, or create derivative works.
- Use the software to provide a service to third parties.
- Use the software to build a competing product or service.

## Maintainer cadence

This is an independently maintained project. Enquiries are handled in scheduled
windows, not continuously — expect a first response within about a week.
"Out of scope" and "won't do, because X" are valid, respectful outcomes.
